Clusterio is an open-source server framework for Factorio that links multiple game servers together into a single "cluster". It adds shared storage and resource transfer across server boundaries, so items you produce on one shard can be deposited into a shared pool and withdrawn on any other shard. This enables large-scale player cooperation that wouldn't be possible on a single server.
Join our Discord server first — server addresses, shard listings, and the required modpack link are posted there. Once you have the modpack installed, open Factorio, go to Multiplayer → Connect to address, and enter the address for the shard you want to join.
