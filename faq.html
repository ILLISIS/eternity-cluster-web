Clusterio is an open-source server framework for Factorio that links multiple game servers together into a single "cluster". It adds shared storage and resource transfer across server boundaries, so items you produce on one instance can be deposited into a shared pool and withdrawn on any other instance. This enables large-scale player cooperation that wouldn't be possible on a single server. You can read more about it on the Clusterio GitHub or the Clusterio Discord.
