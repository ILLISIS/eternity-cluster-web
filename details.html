About the Event

Eternity Cluster is a community-run multiplayer Factorio event built on Clusterio — a framework that links multiple Factorio servers together, allowing players to share resources, items, and electricity across server boundaries. Rather than playing in isolation, every factory you build contributes to a shared cluster goal.


This event is built on a plugin to Clusterio called Gridworld. Gridworld does exactly what you might expect and splits the world into grids, for Eternity those are 1024 x 1024 tiles in size - power, belts, trains etc can all traverse the edges of those grids and onto the next instance like its one great big map. With gridworld we can spread the load of a gigabase over many servers and build a base that is way bigger than the scale which would normally bring your UPS to its knees!

How to Join

  1. Install Factorio — You need a copy of Factorio (version TBD or later).
  2. Join the Discord — Head to our Discord server for the latest announcements, cluster goals, and mod download links.
  3. Connect to the server — Once the event is live, open Factorio, go to Multiplayer → Connect to address, and enter lobby.eternitycluster.com. You will need to synchronise your mods when you first join the server.
  4. Start building! — Start with whatever area you want to build to reach our goal! Not sure what to build? Ask teammates in Discord or in game and they will help you out.

Rules

  • Be respectful to all other players. Harassment will result in a ban.
  • No griefing — do not destroy or sabotage other players' builds.
  • Giving other players advice is fine - but dont be a backseat gamer and mess with their builds unless they ask for help!
  • No cheating, hacking, or use of unapproved mods.
  • Keep the shared storage tidy — do not flood it with unwanted items.
  • Follow any additional rules posted in the Discord event channel.
  • Admins have final say on all disputes and rule interpretations.
