About the Event

Eternity Cluster is a community-run multiplayer Factorio event built on Clusterio — a framework that links multiple Factorio servers together, allowing players to share resources, items, and electricity across server boundaries. Rather than playing in isolation, every factory you build contributes to a shared cluster goal.


Players join one of several shard servers, each representing a different region of the world. Resources can be stored in shared storage and withdrawn on any other shard, enabling true cross-server cooperation and specialisation.

How to Join

  1. Install Factorio — You need a copy of Factorio (version TBD or later). The game is available at factorio.com.
  2. Join the Discord — Head to our Discord server for the latest announcements, shard assignments, and mod download links.
  3. Download the modpack — A link to the required Clusterio modpack will be posted in Discord before the event starts. Install it via the in-game mod portal or by placing files in your mods folder.
  4. Connect to the server — Once the event is live, open Factorio, go to Multiplayer → Connect to address, and enter TBD.
  5. Pick a shard and start building — Choose a server that needs players, or coordinate with teammates in Discord. Resources you produce can be shared with the whole cluster.

Rules

  • Be respectful to all other players. Harassment will result in a ban.
  • No griefing — do not destroy or sabotage other players' builds.
  • No cheating, hacking, or use of unapproved mods.
  • Keep the shared storage tidy — do not flood it with unwanted items.
  • Follow any additional rules posted in the Discord event channel.
  • Admins have final say on all disputes and rule interpretations.
